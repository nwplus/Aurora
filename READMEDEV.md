## Development

### Running the repo for the first time

From the project root directory run

`npm install`

`npm run storybook`

and go to `localhost: 6006`

### Run

`npm run storybook`

and go to `localhost: 6006`

### Production Build

To build your package run:

`npm run build`

To publish to npm (login required)

`npm publish --access public`
